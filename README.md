# AI Workout Tracker

A real-time fitness tracking system built using Computer Vision.

## Features
- Pose detection using MediaPipe
- Squat repetition counter
- Posture correction feedback
- Accuracy tracking system
- UI with real-time visualization

## Tech Stack
- Python
- OpenCV
- MediaPipe
- NumPy
  
## Run Locally

```bash
pip install -r requirements.txt
python app.py
